## 颜色码规范

### 背景色
```
#000000  - 纯黑色（常用场景背景）
#ffffff  - 纯白色
#59d27b  - 绿色背景
```

### 文本颜色（白色系）
```
#ffffff  - 纯白色
#fff7ea  - 浅黄白色
#efebda  - 浅米色
#D0E2E3  - 浅青色
#ecf9f9  - 浅青白色
#e2e2e2  - 浅灰色
```

### 文本颜色（金色系）
```
#f0bd6d  - 金黄色
#efdf87  - 亮金色
#ac7434  - 深金色（确定按钮）
#F2e1b8  - 浅金色
#cd7517  - 橙棕色
```

### 文本颜色（提示色）
```
#f58255  - 橙色（高亮提示）
#3ac98b  - 青绿色（增益数值）
#a7ec91  - 浅绿色
#118843  - 深绿色
```

### 文本颜色（中性色）
```
#6f4f3a  - 棕色（普通文本）
#947e67  - 浅棕色
#947e76  - 灰棕色
#575047  - 深棕灰
#834b29  - 深棕色
#cacaca  - 中灰色
#777777  - 灰色（禁用状态）
```

### 按钮颜色
```
#ac7434  - 确定按钮文字
#20545e  - 取消按钮文字
```

### 描边颜色
```
#000000  - 黑色描边（最常用）
#353535  - 深灰描边
#4a5053  - 灰色描边
#d22f63  - 红色描边
```
