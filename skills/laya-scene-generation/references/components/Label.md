## Label（标签）

显示文本，支持丰富样式。

**常用属性**:
```json
{
    "type": "Label",
    "props": {
        "x": 0,
        "y": 0,
        "var": "lb_title",
        "text": "标题文本",
        "fontSize": 40,
        "color": "#ffffff",
        "stroke": 2,
        "strokeColor": "#000000",
        "align": "center",
        "valign": "middle",
        "wordWrap": true,
        "width": 300,
        "height": 50,
        "leading": 10,
        "bold": false,
        "italic": false
    }
}
```
