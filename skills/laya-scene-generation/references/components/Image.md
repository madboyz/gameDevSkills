## Image（图像）

显示图片资源。

**常用属性**:
```json
{
    "type": "Image",
    "props": {
        "x": 0,
        "y": 0,
        "var": "img_bg",
        "skin": "res/oldpic/confirm/yt_panel_10.png",
        "sizeGrid": "23,32,16,37",  // 九宫格切片
        "width": 200,
        "height": 200,
        "visible": true,
        "anchorX": 0.5,
        "anchorY": 0.5
    }
}
```

**九宫格格式**: `"上边距,右边距,下边距,左边距"`
