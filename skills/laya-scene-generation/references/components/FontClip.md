## FontClip（位图字体）

使用图片显示数字或字符。

**常用属性**:
```json
{
    "type": "FontClip",
    "props": {
        "x": 0,
        "y": 0,
        "var": "num_clip",
        "value": "12345",
        "skin": "res/public/img_zlnumxiao.png",
        "sheet": "0123456789.abcd",
        "spaceX": -2,
        "align": "center"
    }
}
```
